# 🚗 DIGIVAHAN-iOS

DIGIVAHAN-iOS is the official native iOS application developed for DigiCapital, designed to provide users with a seamless and secure digital vehicle and financial services experience.

The application enables users to access vehicle-related information, loan services, credit score insights, EMI calculators, document management, and other financial solutions through a modern and intuitive interface.

## ✨ Features

- 🚗 Vehicle Information
- 📄 RC Verification
- 💳 Credit Score (CIBIL) Check
- 📈 Credit Score Improvement Assistance
- 💰 Loan Eligibility Calculator
- 🧮 EMI Calculator
- 📂 Document Upload & Management
- 🔔 Real-time Notifications
- 🤖 AI Financial Assistant
- 👨‍💼 Personal Financial Advisor Support
- 🔒 Secure Authentication
- 📱 Clean & Modern User Experience

## 🛠️ Tech Stack

- Swift
- SwiftUI / UIKit
- MVVM Architecture
- URLSession / Alamofire
- Codable
- Keychain
- UserDefaults
- REST APIs
- Apple Push Notification Service (APNs)

## 🚀 Getting Started

1. Clone the repository.
2. Open `DIGIVAHAN-iOS.xcodeproj` or `DIGIVAHAN-iOS.xcworkspace`.
3. Install dependencies (if applicable).
4. Configure the required environment variables and API endpoints.
5. Build and run the application using Xcode.

## 🔐 Security

- Secure API communication using HTTPS.
- Sensitive information stored securely using Keychain.
- Token-based authentication.
- Industry-standard security practices.
